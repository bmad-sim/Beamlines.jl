## Overview

Beamlines.jl is designed around a few core concepts:

- **LineElements**: Polymorphic, extensible element types
- **Beamlines**: Collections of elements with reference energy
- **Deferred Expressions**: Lazy evaluation for dependent quantities
- **Controllers**: Coordinated control of multiple elements

Each of these topics is covered in detail in the following sections.
